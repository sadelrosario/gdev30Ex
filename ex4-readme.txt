It is recommended to hold the key to see the changes in the scene more visibly.

LIGHT POSITION CONTROLS:
up arrow key = move light position upwards
down arrow key = move light position downwards
left arrow key = move light position to the left
right arrow key = move light position to the right
Z key = increase light position by the z-axis (move it forward)
X key = decrease light position by the z-axis (move it backwards)

SPECULAR / SHININESS CONTROLS:
Q key = decrease specularity value
E key = increase specularity value
